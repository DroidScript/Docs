Creates a circular game object.

cir = gfx.CreateCircle( width, color, lineWidth, lineColor, lineAlpha, group ) → app object: Circle

Example

cfg.Game;

function OnLoad()
{
    circle = gfx.CreateCircle( 0.4, 0xff1133, 6, 0x7799ff );
}

function OnReady()
{
    gfx.AddGraphic( circle, 0.3, 0.2 );
    gfx.Play();
}
