AddText

Hello World

Adds a text object to the game.

gfx.AddText( obj, x, y, angle, alpha )

Example

cfg.Game;

function OnLoad()
{
    text = gfx.CreateText( "Hello World", 40, "/Sys/Img/Desyrel.xml" );
}

function OnReady()
{
    gfx.AddText( text, 0.2, 0.4, 0.2 );
    gfx.Play();
}
