ZipFile

Hello World

Compresses a local file to a zip file.

app.ZipFile( source, destination, options )

See Also: ZipFolder

Example - Zip File

function OnStart()
{
    var file = "/sdcard/DroidScript/Hello World/Hello World.js";
    app.ZipFile( file, file + ".zip" );
    app.ShowPopup( "Compressed to " + file + ".zip" )
}
