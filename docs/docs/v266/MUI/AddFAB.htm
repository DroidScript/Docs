AddFAB

Hello World

Adds a floating action button.

fab = MUI.AddFAB( parent, icon, color, options ) → Object

See CreateFAB, CreateFABElegant and CreateFABOutline for a full documentation.

Example - AddFAB

cfg.Light
cfg.MUI

function OnStart()
{
    lay = MUI.CreateLayout("Linear", "VCenter,FillXY")

    fab1 = MUI.AddFAB(lay, "favorite", null, "default")
    fab2 = MUI.AddFAB(lay, "favorite", MUI.colors.deepPurple.lighten1, "outline")
    fab3 = MUI.AddFAB(lay, "favorite", MUI.colors.teal.darken1, "elegant")

    app.AddLayout(lay)
}
