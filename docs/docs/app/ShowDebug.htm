ShowDebug

Hello World

Show/Hide the debug console as overlay in your app, after calling CreatgeDebug.

app.ShowDebug( show )

See Also: Debug

Example - Toggle Debug

function OnStart()
{
    app.CreateDebug();

    lay = app.CreateLayout( "linear", "VCenter,FillXY" );

    tgl = app.CreateToggle( "WiFi Access Point", -1, 0.1 );
    tgl.SetOnTouch( ShowDebug );
    tgl.SetChecked( true );
    lay.AddChild( tgl );

    app.AddLayout( lay );
}

function ShowDebug( show )
{
    app.ShowDebug( show );
}
