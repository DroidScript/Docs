AddLayout

AddLayout adds a layout to the screen so that it is visible (if not hidden with SetVisibility).

app.AddLayout( layout )

Example - Vertical

function OnStart()
{
    lay = app.CreateLayout( "Linear", "FillXY,VCenter" );

    txt = app.CreateText( "Hello" );
    txt.SetTextSize( 50 );
    lay.AddChild( txt );

    app.AddLayout( lay );
}
