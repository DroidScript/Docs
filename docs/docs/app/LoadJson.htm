LoadJson

Hello World

Load a saved JSON object use it as a varibale.

app.LoadJson( name, default, file ) → Object

See also: SaveJson.

Example - First Time

var file = "demofile";

function OnStart()
{
    var settings = app.LoadJson( "IsFirst", {"IsFirst":false}, file );

app.Alert( JSON.stringify( settings) );
    app.SaveJson( "IsFirst", {"IsFirst":true}, file );
}
