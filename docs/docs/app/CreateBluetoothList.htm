CreateBluetoothList

CreateBluetoothList is an Android dialog which allows user to select a Bluetooth device from paired and discovered devices.

btl = app.CreateBluetoothList( filter ) → app object: BluetoothList

You can also use GetPairedBtDevices and DiscoverBtDevices.


The following methods are available on the BluetoothList object:

GetType() → string: “BluetoothList”
SetOnTouch( callback )

unknown
Returns the control class name.
function(name, address)
Define a callback function that is called when the user touches the control.
