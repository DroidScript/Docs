GetSSID

Hello World

Returns the name (SSID) of the current network the user is connected with.

app.GetSSID() → string

If WiFi is disabled “” is returned.

See Also: IsWifiEnabled, SetWifiEnabled, WifiConnect

Example - Show SSID

function OnStart()
{
    var ssid = app.GetSSID();
    app.Alert( ssid );
}
