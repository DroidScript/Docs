GetInstalledApps

Hello World

Returns a list of objects containing informations about installed apps.

app.GetInstalledApps() → list: [{ packageName, className, uid, targetSdkVersion, dataDir, sourceDir, publicSourceDir, nativeLibraryDir }]

See Also: GetRunningApps, GetRunningServices

Example - Show installed apps

function OnStart()
{
    app.ShowProgress();

    var lay = app.CreateLayout( "linear" );

    var list = app.GetInstalledApps();

    lst = app.CreateList( "", 1, 1 );
    lay.AddChild( lst );

    for(var i in list) {
        var a = list[i];
        lst.AddItem(a.packageName);
    }

    app.AddLayout(lay);

    app.HideProgress();
}
